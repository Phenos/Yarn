Punctuation

Commas, periods and semi-colons.

By adding puncutation, you can create series of assertions that are both easier to write and easier to read.

Here is an example of how you can simplify longer texts by using commas.

Peter The Dog is a Dog. Peter The Dog is in the Kennel. Peter The Dog is Tired.

Can be shortened to:

Peter The Dog is a Dog, is in the Kennel, is Tired.

Just like a normal english sentence, the commas allow you to chain more assertions without having to repeat the subject.

Adding linefeeds

To keep your text more readable, you can cut you line of text inbetween the 3 parts of the assertions without any side-effects

Peter The Dog is a Dog, is in the Kennel, is Tired

Indentation

You can also use indentation (additionnal spaces) if you want to give more structure to your code. This is purely aesthetics and will not change the behavior of the code.

Let's add some to Peter The Dog:

Peter The Dog is a Dog, is in the Kennel, is Tired

Using quotes for texts and labels

For labels and long texts you simply use either the single or double quotes. Like this:

Peter The Dog is a Dog, has a Name "Peter T. Dog", is in the Kennel, has a Description 'Peter your dog has had a long day. He looks very tired'

Notice how you can also add line breaks to break long texts.

Quotes inside quotes

If you need to use quotes in your text, you have to be careful not to use the same type of quotes you used to start the text block. For example, these examples are valid:

Dog has a Description "This is a 'great' dog!". Dog has a Description 'This is a "great" dog!'.

But these examples will break your text:

Dog has a Description "This is a "great" dog!". Dog has a Description 'This is a 'great' dog!'.

Also, you can use the html "&quot;" symbol for using double quotes inside double quotes. So this example is valid:

Dog has a Description "This is a &quot;great&quot; dog!". Dog has a Description 'This is a &quot;great&quot; dog!'.

Add periods to prevent chaining.

Whenever you have assertions which precede or follow each others, you can make sure they dont automatically chain by adding a period.

Here we make sure John and PeterTheDog dont get mixed up:

John loves Peter The Dog. Peter The Dog is a Dog, has a Name "Peter T. Dog".

Alternativelly, you can use two line breaks instead of periods. Like so:

John loves Peter The Dog Peter The Dog is a Dog, has a Name "Peter T. Dog" Silly Susan loves John
