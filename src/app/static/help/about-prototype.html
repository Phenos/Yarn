About this prototype

It is still incomplete!

Yarn Studio is still under developement on a daily basis. What you are using is a "work in progress". Major features are still coming.

Keep in mind that:

  • It might be missing features you would expect it to have.
  • You might encounter bugs.
  • The documentation is also growing weekly, but incomplete
  • The code you write today might need to change as we improve the platform.

But, it's already good for showing:

  • The overall concept of the application
  • The kind of user interfaces and user experience to expect
  • The relationship between the player, the editor and the console
  • The gameplay experience
  • The performance (speed is similar to expected end product)
  • How the language look and works in general
  • How the game world behaves in relation with the code you write

Dont hesitate to:

  • Follow us on twitter @YarnStudioGames
  • Read our development blog
  • Recommend us features and ideas
