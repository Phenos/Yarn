For Authors

Don't worry, it's not Rocket Science

If you are wondering if you need to be a computer programmer to write interactive stories with Yarn, don't worry.

It does require you to learn new concepts and a specialised language, but know that these tools have been made for people with "very basic" coding skills. Most computer literate people can learn the basics in a few hours.

Things can get wild as you try to create more complex stories, but we try to make the core features as easy to use as possible.
