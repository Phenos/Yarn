Comments

Sometimes it is usefull to add comments to your source code in order to better explain what your code is supposed to do or keep track of all the ideas you have while writing.

Line comments

You can add single-line comment by using the double slash "//" prefix.

// Note: Peter's details are described later in the "characters" file Peter The Dog is a Dog, has a Name "Peter T. Dog" // Not sure about this name Susan has a Name "Silly Susan".

Block comments

For longer section of comments you can use the /* */ delimiters to create comments blocks (multiple lines)

John loves Peter The Dog /* I have to remember to add a more detailed description of peter. Also dont forget to add the backstory for John. */ Peter The Dog is a Dog, has Name "Peter T. Dog".

Use them anywhere

Comments can be added elsewhere than at the beginning of lines. Like this:

John loves Peter The Dog // Maybe too much Peter The Dog is a Dog /* Obviously */, has Name "Peter T. Dog".

Commenting out statements

Remember that comments are also useful for disabling assertions whenever you testing your story or to keep track of multiple ideas

Peter The Dog is a Dog, has Name "Peter T. Dog". //John loves Peter The Dog John loves Susan.
