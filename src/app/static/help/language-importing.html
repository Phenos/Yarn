Importing

Most interactive stories will quickly grow into files with thousands of lines of code. The more thay grow the more unmanageable and confusing the become.

So it is a good idea to separate your stories into multiple files. And then import them from your main story file.

The "import" keyword

Here is an example with 3 files. A main file using the "import" keyword to import the other two.

story.txt

Story has Name "John's crummy bedroom". import "characters.txt". import "bedroom.txt".

characters.txt

John is an Actor, named "John Johnson", is in the Bedroom.

bedroom.txt

Bedroom is a Place, called "The Bedroom".
