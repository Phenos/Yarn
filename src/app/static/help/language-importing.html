Importing

More interactive stories will quickly grow into files with thousands of lines of code. The more thay grow the more unmanageable and confusing the become.

So it is a good idea to separate your stories into multiple files. And then import them from your main story file.

The @import keyword

Here is an example with 3 files. A main file using the "@import" keyword to import the other two.

story.yarn.txt

Story is called "John's bedroom". @import "characters.yarn.txt". @import "bedroom.yarn.txt".

characters.yarn.txt

John is an Actor, named "John Johnson", is in the Bedroom.

bedroom.yarn.txt

Bedroom is a Room, called "The Bedroom".
