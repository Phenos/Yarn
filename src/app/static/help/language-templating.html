Templating

Whenever text is outputed to the players screen, yarn will first check if you specified that you text is a template that need to be evaluated first.

Templating is usefull for:

  • Vary text blocks according to conditions
  • Insert values in text such as names, numbers, time and other values.
  • Outputing dynamic lists of items

To specified that a text is a template, you precede it with tmpl:, like in this example:

MagicHat has aDescription "tmpl: Your trusty pointy hat. {% if assert('You are in Castle') %} Too large and flappy as usual. {% else %} Weirdly it is glowing octarine... you must be near the magic shop!. {% endif %}"

In the example, the hats description changes depending on the user's location.

The Nunjucks Templating Engine

This templating language is called "Nunjucks" and is used by many other projets. A more extensive and generic purpose documentation is available on their website at:
http://mozilla.github.io/nunjucks/templating.html
