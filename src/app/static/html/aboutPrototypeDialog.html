About this prototype

It is still incomplete!

Yarn Studio is still under developement on a daily basis and what you are using is a only a prototype for demonstration purposes.

Keep in mind that:

  • It is not a complete and finished product. You might encounter bugs.
  • The documentation is also incomplete
  • The code you write in the editor might be deleted/lost in future versions of the servive.
  • While writing the story, you will not get a lot of hint on compilation errors

But, it's a good demonstration of:

  • The overall concept of the application (web app, mobile)
  • The kind of user interfaces and user experience to expect
  • The relationship between the player, the editor and the console
  • The gameplay experience on the player/reader
  • The performance (speed is similar to expected end product)
  • How the language look and works in general
  • How the world behaves in relation with the code you write and the gameplay

Dont hesitate to:

  • Follow us on twitter @YarnStudioGames
  • Read our development blog
  • Recommend us features and ideas

The development is expected to continue troughout 2016 and fully stable and usable version of the service to be released by end of 2016.
